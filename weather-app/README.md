# 🌤️ Weather App (React + OpenWeather API)

A modern and responsive weather application built using React.js that fetches real-time weather data and displays it with dynamic UI effects.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* 🌥️ Dynamic weather conditions (Clear, Clouds, Rain, etc.)
* 🎨 Animated UI effects inside weather card
* 💨 Wind speed and humidity data
* ⚠️ Error handling for invalid city
* ⏳ Loading state feedback

---

## 🛠️ Tech Stack

* React.js
* CSS (Glassmorphism + Animations)
* OpenWeatherMap API

---

## 📦 Installation

```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
npm install
npm run dev
```

---

## 🔑 API Setup

Replace API key inside App.jsx:

```js
appid=YOUR_API_KEY
```

Get API key from:
https://openweathermap.org/api
