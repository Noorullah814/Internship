# 🚀 React Premium Components Library

## 📌 Project Description
This project is a reusable React component library that includes commonly used UI components such as Buttons, Cards, and Modals. The goal of this project is to demonstrate component-based architecture, props handling, and modern UI design practices.

The components are designed to be flexible, reusable, and easy to integrate into any React application.

---

## ✨ Features

- 🔘 Multiple Button Variants (Primary, Secondary, Outline, Ghost)
- 📏 Different Button Sizes (Small, Medium, Large)
- 🃏 Reusable Card Component (with image & content support)
- 🪟 Modal Component with overlay and animations
- 🎨 Modern UI with smooth transitions and hover effects
- 📱 Fully Responsive Design
- ♻️ Reusable and clean component structure

---

## 🛠️ Technologies Used

- React.js (Vite)
- JavaScript (ES6+)
- CSS3 (Flexbox, Grid, Animations)
- Google Fonts (Poppins)

## ⚙️ Installation & Setup

1. Clone the repository: git clone <your-repo-link>


2. Navigate to project folder: cd react-components-library


3. Install dependencies: npm install


4. Run the project: npm run dev
