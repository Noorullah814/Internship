# React Counter & Form Handling App

## Project Description
This project is a simple React application that demonstrates core React concepts such as state management, props, event handling, and form handling.

The application contains two main components:

1. Counter Component
2. Contact Form Component

The counter allows users to increment, decrement, and reset a number value.

The contact form collects user information including name, email, and message. After successful submission, a confirmation message is displayed.

This project was developed as part of a React internship assignment.

---

## Features

- React Functional Components
- useState Hook
- Props for data passing
- Event handling
- Form handling
- Input validation
- Conditional rendering
- Responsive UI design

## Installation

1. Clone the repository
git clone https://github.com/https://github.com/Noorullah814/Internship/react-counter-form-app.git

2. Navigate to project folder


cd react-counter-form-app


3. Install dependencies


npm install


4. Start the development server


npm run dev


---

## Usage

- Use the counter buttons to increase, decrease, or reset the counter value.
- Fill out the contact form with your name, email, and message.
- After submission, a success message will appear confirming the form submission.
